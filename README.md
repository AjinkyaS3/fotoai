# fotoai
A Assignment given by the Company
Technologies Used
Backend

Python – Core backend logic and API development

FastAPI – Building RESTful APIs for image import and retrieval

Uvicorn – ASGI server for running FastAPI applications

Google Drive API – Fetching images from public Google Drive folders

SQL Database (e.g., PostgreSQL / SQLite) – Storing image metadata

Environment Variables (.env) – Secure configuration management

Background Worker

Python Worker Service – Handles image processing and uploads asynchronously

Queue-based Processing – Designed to support large-scale image imports

Dockerized Worker – Runs independently for scalability

Frontend

HTML5 – User interface structure

CSS3 – Styling and layout

JavaScript (Vanilla JS) – Client-side logic and API interaction

Nginx – Serving frontend and reverse proxy configuration

Containerization & Deployment

Docker – Containerization of backend, frontend, and worker services

Docker Compose – Multi-service orchestration and local environment setup

Configuration & Tooling

Nginx Configuration – Routing and frontend hosting

Git & GitHub – Version control and source code management
